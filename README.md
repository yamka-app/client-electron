## Build
Do `npm start` to run the app without producing any artifacts:
```sh
git clone https://github.com/yamka-app/client-electron.git
cd client-electron
npm i
npm start
```

Use this to produce `AppImage` and `exe` artifacts:
```sh
git clone https://github.com/yamka-app/client-electron.git
cd client-electron
npm i
npm run dist
```
